# FlowUIX

An HTML-first, zero-JavaScript UI framework built with TailwindCSS.

## What is FlowUIX?

FlowUIX is a lightweight UI component library that ships **100% CSS-only**. Perfect for HTMX, Alpine.js, and server-driven apps â€” no JavaScript required in the core.

**Key Features:**
- Zero JavaScript components
- HTML-first & framework-agnostic
- Built-in dark/light themes
- AI-friendly class naming
- Ultra-lightweight TailwindCSS plugin

## Quick Example

```html
<div class="fx-card p-6">
  <h2 class="fx-card-title">Hello FlowUIX</h2>
  <p class="text-gray-600">Pure HTML and CSS components.</p>
  <button class="fx-btn fx-btn-primary mt-4">Get Started</button>
</div>
```
